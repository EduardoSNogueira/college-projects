# Library System

## 📝 Description
The **Library System** is a Python-based application designed to automate library management processes. The project manages book registration, user control, and the workflow for loans and returns in an efficient and structured manner.

This project is part of my academic portfolio and focuses on the practical application of programming logic and data manipulation.

## 🚀 Technical Objectives
* Implementation of **CRUD** operations (Create, Read, Update, Delete).
* Data persistence (initially via structured JSON files).
* State management and handling of complex conditional flows.
* Practical application of code modularization in Python.

## 🏗 Project Structure
* `main.py`: Application entry point.
* `src/`: Core business logic modules (books, users, loans).
* `data/`: Storage for system data files.
* `.venv/`: Isolated virtual environment for dependencies.

## 🛠 Tech Stack
* **Python 3.12+**
* **Venv**: Virtual environment for package isolation.
* **Ruff/Pylance**: Linting and code quality assurance tools.
